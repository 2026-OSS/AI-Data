# YOLO11n v14 Artifacts

Place the trained YOLO11n v14 handoff files in this directory.

Tracked handoff files:

- `best.pt`: trained model weights for backend inference
- `data.yaml`: dataset class names and paths used during training
- `results/`: evaluation summaries and review graphs
- `predictions/`: saved test prediction images

Other temporary training outputs should stay out of git.
